## ‚ùî About

<p align="justify"><i>React Firebase-Chat</i> is a chat or messaging app build with the help of Firebase, which is a Backend-as-a-Service (BaaS) owned by Google. Firebase has extensive features for building, hosting and managing apps but (at this point) I am mainly using it for user Authentication, file Storage and as a Database.</p><br/>

 <p>Example <i>.env</i> file (based on your project's Firebase configuration)</p>

```sh
REACT_APP_API_KEY=""
REACT_APP_AUTH_DOMAIN=""
REACT_APP_AUTH_PROJECT_ID=""
REACT_APP_STORAGE_BUCKET=""
REACT_APP_MESSAGING_SENDER_ID=""
REACT_APP_APP_ID=""
REACT_APP_MEASUREMENT_ID=""
```
